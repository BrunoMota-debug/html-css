Sobre este site

Este site foi criado para demonstrar o uso de cores e gradientes em CSS. O fundo é um degradê que vai do vermelho ao roxo, criando um efeito visual interessante.

Você pode personalizar as cores do degradê alterando os valores no código CSS. Experimente diferentes combinações para ver como elas ficam!
